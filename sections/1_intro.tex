	\section{Introduction}
	%\cite{Bowker1985}
	\hspace{\parindent}A Bloom filter is a data structure that utilizes hashing and a bit array to determine what elements have definitely not been inserted into it. Because the inclusion in the set is only represented as a bit array, the space to represent a set is greatly reduced. The time it takes to insert and check large datasets can be mitigated by parallelizing the insertion and checking steps.%
